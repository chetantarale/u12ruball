u12ruball
================

Shippable CI image for Ruby on Ubuntu 12.04 with all services. Available versions are:

1. 1.8.7
2. 1.9.3
3. 2.0.0
4. 2.1.5
5. 2.2.1

**Services:**

1. cassandra 2.1
2. elasticsearch 1.5
3. memcached 1.4
4. mongodb 3.0
5. mysql 5.6
6. neo4j 2.2
7. postgres 9.4
8. rabbitmq 3.5
9. redis 3.0
10. selenium 2.45
11. sqllite 3
12. couchdb 1.6
13. rethinkdb 2.0
14. riak latest
